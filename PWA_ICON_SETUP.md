# PWA アイコン作成手順

このアプリをPWA（Progressive Web App）として完全に機能させるには、アイコン画像が必要です。

## 必要なファイル

以下の2つのアイコンファイルを `public/` ディレクトリに配置してください：

### 1. icon-192.png
- サイズ: 192x192 ピクセル
- 形式: PNG
- 用途: スマホのホーム画面アイコン

### 2. icon-512.png
- サイズ: 512x512 ピクセル
- 形式: PNG
- 用途: スプラッシュスクリーンなど

## 簡単な作成方法

### オプション1: オンラインツール
1. https://www.favicon-generator.org/ にアクセス
2. 任意の画像をアップロード
3. 192x192 と 512x512 のサイズでダウンロード

### オプション2: 画像編集ソフト
1. 512x512 の正方形画像を作成（カラー: #3b82f6 推奨）
2. 中央に「文」または「文芸部」のテキストを配置
3. 512x512 で保存 → icon-512.png
4. 192x192 にリサイズして保存 → icon-192.png

### オプション3: テンポラリアイコン（開発用）
単色の正方形画像でも動作します：
1. ペイントなどで 512x512 の青い正方形を作成
2. icon-512.png として保存
3. 192x192 にリサイズして icon-192.png として保存

## 配置場所

```
public/
  ├── icon-192.png  ← ここに配置
  ├── icon-512.png  ← ここに配置
  ├── manifest.json
  └── sw.js
```

## 確認方法

1. アイコンを配置後、アプリを開く
2. ブラウザのアドレスバー右に「インストール」ボタンが表示される（Chrome）
3. スマホで開いた場合、「ホーム画面に追加」が利用可能になる

## 注意事項

- アイコンがないとPWAとして認識されない場合があります
- 正方形の画像を使用してください
- 透過PNG も使用可能ですが、背景色を設定することを推奨します
